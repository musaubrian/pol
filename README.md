# POL

naive reverse polish notation

## Setup

```sh

git clone github.com/musaubrian/pol

cd pol

go build .
```

## Usage

```sh
./pol -h
```

- [x] Single Expression
- [ ] Grouped Expressions
     - [x] 2 Groups
     - [ ] Nested grouped expressions
- [x] file input expression evaluation
- [x] REPL
